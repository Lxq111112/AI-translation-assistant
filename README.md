# AI 智能翻译与摘要助手

这是一个基于 Streamlit 构建的轻量级 Web 应用，能够帮助用户将输入文本进行翻译、摘要，或同时完成“翻译 + 摘要”处理。

项目通过调用火山引擎 Ark 大模型接口，实现自然语言处理能力，适合用于日常文本处理、内容理解和信息提炼。

## 功能特点

- 支持三种处理模式：
  - 翻译
  - 摘要
  - 翻译 + 摘要
- 支持多种目标语言选择：英文、中文、日文、韩文、法文、德文、西班牙文
- 支持调整摘要长度
- 提供简洁的 Web 界面，操作方便
- 处理结果可直接在页面中展示

## 项目结构

```text
Translate_Assistant/
├── app.py             # Streamlit 前端界面
├── ai_service.py      # 调用大模型的业务逻辑
├── .env               # 环境变量文件（本地配置）
├── .gitignore         # Git 忽略文件
└── README.md          # 项目说明文档
```

## 主要文件说明

- app.py
  - 负责构建用户界面
  - 提供文本输入框、操作类型选择、按钮和结果展示
  - 通过调用后端函数完成处理

- ai_service.py
  - 封装了翻译、摘要以及联合处理逻辑
  - 使用 OpenAI SDK 与火山引擎 Ark API 进行交互
  - 通过环境变量读取 API Key，并配置模型参数

## 环境要求

- Python 3.9+
- 依赖包：
  - streamlit
  - openai
  - python-dotenv

可以通过以下命令安装依赖：

```bash
pip install streamlit openai python-dotenv
```

## 配置说明

在项目根目录下创建 `.env` 文件，并填入你的 API Key：

```env
ARK_API_KEY=你的火山引擎 Ark API Key
```

> 该项目当前使用了火山引擎 Ark 的接口地址和模型配置，需确保你拥有有效的 API 权限。

## 运行方式

在项目根目录执行：

```bash
streamlit run app.py
```

运行后，浏览器会打开应用页面，您可以直接开始使用。

## 使用流程

1. 打开应用页面
2. 在输入框中粘贴或输入待处理文本
3. 选择处理方式：翻译 / 摘要 / 翻译 + 摘要
4. 如需翻译，选择目标语言
5. 如需摘要，调整摘要长度
6. 点击“开始处理”按钮
7. 查看生成结果

## 注意事项

- 本应用调用大模型 API，处理速度和结果质量取决于网络环境和模型服务状态
- 生成结果仅供参考，实际使用时请结合上下文判断
- 请妥善保管 `.env` 文件，避免将 API Key 泄露
